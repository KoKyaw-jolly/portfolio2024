Human Resource Management System(HRMS)

The Human Resources Management System (HRMS) is a web application designed to streamline HR operations, making it easier for businesses to manage employee records, leave requests, and access control based on roles. This project focuses on improving HR efficiency through an intuitive, user-friendly interface.

Note: In the live demo, all pages are linked with the wireframe. All form elements have actions (hover, click). To add a value in a textbox, click and press the “V” key. To delete the value, press the “Esc” key.

GitHub Repository

This repository contains the entire frontend of the HRMS built using Angular 18. It utilizes Ant Design (ng-zorro) for UI components and NgRx for state management. Please note, only the frontend has been developed. You can set up and test the project by following the instructions in the README file, which also provides steps for running the mock backend using Mockoon.

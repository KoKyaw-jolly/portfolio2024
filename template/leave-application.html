Leave Application System (Web App)

The Leave Application System is a web-based solution designed to manage employee leave requests, holiday management, and notifications. It helps streamline the process of applying for leaves, generating reports, and keeping track of company holidays. The system is built with a focus on ease of use and efficient management for both employees and administrators.

GitHub Repository

This repository contains the entire frontend of the Leave Application System built using Angular 18. It utilizes Ant Design (ng-zorro) for UI components and NgRx for state management. The project is front-end focused, and the backend is simulated using Mockoon. You can find the setup and testing instructions in the README file, which includes steps to run the mock backend.

Key Features
  • Apply for leave, View leave reports, Manage holidays and notifications
  • Frontend developed using Angular 18 with a focus on performance and scalability
  • UI components built with Ant Design System (ng-zorro)
  • NgRx used for state management
